# easy-flow

> 基于VUE+JsPlumb的流程设计器

![](https://github.com/BiaoChengLiu/easy-flow/blob/master/src/assets/easy-flow.png?raw=true)

## 功能
* 支持拖拽添加节点
* 鼠标移入到节点中显示 编辑、删除 操作
* 不支持节点线连接自己
* 不支持节点A与节点B之间线互连

## 启动

``` bash
# 下载工程
git clone  https://github.com/BiaoChengLiu/easy-flow.git

# 安装依赖包
npm install

# 启动
npm run dev

# 访问地址
 http://localhost:8080
```
