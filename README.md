# easy-flow

> 基于VUE+JsPlumb的流程设计器

![](https://github.com/BiaoChengLiu/easy-flow/blob/master/src/assets/easy-flow.png?raw=true)

## 启动

``` bash
# 下载工程
git clone  https://github.com/BiaoChengLiu/easy-flow.git

# 安装依赖包
npm install

# 启动
npm run dev

# 访问地址
 http://localhost:8080
```
